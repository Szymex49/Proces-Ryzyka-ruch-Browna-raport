# Proces Ryzyka i ruch Browna
 
### Raport przygotowali:
Kacper Budnik <br> Szymon Malec
 
### Opis
Raport składa się z dwóch zadań.

Pierwsze polegało na przeanalizowaniu danych opisujących 50 trajektorii pewnego procesu Ryzyka w celu wyznaczenia jego parametrów. Wymagało to między innymi znalezienia rozkładów strat oraz czasów oczekiwania pomiędzy kolejnymi stratami. Gdy już udało nam się tego dokonać, wykorzystaliśmy wyznaczony model do oszacowania prawdopodobieństwa ruiny za pomocą metody Monte Carlo.

Celem drugiego zadania było oszacowanie średniego czasu wyjścia ruchu Browna z zadanego przedziału w zależności od punktu startowego.
